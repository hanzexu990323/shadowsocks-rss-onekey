# shadowsocks-rss-onekey

A one key script to deploy [shadowsocks-rss](https://github.com/breakwa11/shadowsocks-rss) easier.

## Install
`git clone https://github.com/mikelei8291/shadowsocks-rss-onekey.git`
`bash install.sh`
